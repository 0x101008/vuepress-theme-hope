```md
- Home icon: <FontIcon icon="home" />

```

## Props

### icon

- Type: `string`
- Required: Yes

Icon name

### color

- Type: `string`
- Default: `'inherit'`

Color used for icon.

### size

- Type: `number | string`
- Default: `Current font size`

Icon size.
